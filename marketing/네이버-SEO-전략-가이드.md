# 네이버 블로그 SEO 전략 가이드 — koreatide.com 홍보용

목표: 네이버 검색에서 "물때", "만조 간조", "○○ 갯벌" 류 키워드로 블로그 글이 노출되고,
그 글에서 **koreatide.com 방문**으로 이어지게 만드는 것.

---

## 1. 네이버 검색 로직 한 줄 요약

네이버는 구글과 다릅니다. **C-Rank**(출처의 신뢰도·전문성)와 **D.I.A / DIA+**(문서 자체의
경험·정보성)로 순위를 매깁니다. 핵심은 세 가지입니다.

1. **한 주제로 꾸준히** 쓴 블로그일수록 그 주제에서 잘 밀어줍니다 → "바다/낚시/여행" 축을 유지.
2. **직접 경험**(사진, 후기, 구체적 시간·장소)이 담긴 글을 좋아합니다 → AI 티 나는 글 지양.
3. **체류 시간·클릭**이 중요합니다 → 제목 낚시가 아니라 실제로 끝까지 읽게 만드는 구성.

## 2. 키워드 전략

### 메인 키워드 (검색량 큼, 경쟁도 높음)
`물때`, `물때표`, `만조 간조`, `만조 시간`, `간조 시간`, `조석표`, `물때 조회`

### 롱테일 키워드 (검색량 중, 전환·노출 유리) ← **여기부터 공략**
- `[지역]+ 물때표` : 대부도 물때표, 안면도 물때표, 대천 물때표, 영종도 물때표, 을왕리 물때
- `[지역]+ 갯벌 체험 시간`, `[지역]+ 조개잡이 물때`
- `[지역]+ 바다낚시 물때`, `서해 낚시 물때`, `갯바위 물때`
- `해루질 물때`, `사리 조금 뜻`, `물때 보는 법`

> 전략: 경쟁이 센 `물때` 단독보다 **지역명을 붙인 롱테일**부터 1등을 여러 개 만드세요.
> 지역 글이 쌓이면 블로그의 "바다 전문성"이 올라가 메인 키워드도 따라 올라갑니다.

### 키워드 확인 도구 (무료)
- **네이버 검색광고 > 키워드 도구** (searchad.naver.com): 월 검색량·경쟁도 확인
- 네이버 검색창 **자동완성**과 하단 **연관검색어**: 실제 사람들이 치는 표현 그대로 사용

## 3. 제목 공식

`[지역/상황] + [핵심키워드] + [혜택/행동]`

- 좋은 예: `대부도 갯벌 체험 시간, 물때표 이렇게 확인하세요`
- 좋은 예: `안면도 낚시 물때 (만조·간조 시간 한눈에)`
- 피할 것: `대박 꿀팁 물때 총정리!!!` (과장·특수문자 남발 → 저품질 위험)

규칙: **핵심키워드를 제목 앞쪽**에 두기 / 특수문자·이모지 남발 금지 / 25~35자 내외.

---

## 4. 본문 작성 규칙 (D.I.A 최적화)

- **길이**: 공백 제외 **1,500자 이상** 권장. 단, 억지로 늘리지 말고 정보로 채우기.
- **사진**: **본인이 찍은 사진 5장 이상**. 갯벌·바다·현장 + koreatide.com 화면 캡처.
  (타 사이트 이미지 퍼오기 금지 — 저품질/저작권 위험)
- **키워드 밀도**: 메인 키워드를 제목·첫 문단·소제목·마지막 문단에 자연스럽게 3~5회.
  같은 단어 도배는 역효과.
- **구조**: 소제목(굵게)으로 문단 나누기 → 모바일 가독성↑ → 체류시간↑.
- **직접 경험 서술**: "저는 간조 1시간 전에 도착했어요" 같은 구체적 문장이 DIA+에 유리.
- **외부링크**: koreatide.com 링크는 본문에 **1~2회**면 충분(과하면 스팸 신호). 텍스트로
  `https://koreatide.com` 를 그대로 노출하면 클릭·인지에 유리.

## 5. 발행·운영 전략

| 항목 | 권장 |
|---|---|
| 발행 주기 | 주 2~3회, **같은 주제(바다/물때/낚시/여행)** 로 꾸준히 |
| 발행 시간 | 타깃이 검색하는 시간(주말 나들이 전날인 **금요일 저녁**, 평일 점심) |
| 시리즈화 | "전국 물때표 시리즈" 로 지역별 1편씩 → 내부 서로 링크로 연결 |
| 서로이웃 | 여행·낚시·캠핑 블로거와 서로이웃 → 초기 유입·체류 신호 |
| 재방문 유도 | 글 끝에 "koreatide.com 즐겨찾기 해두세요" 로 사이트 재방문 습관화 |

### 첫 4주 예시 계획
1주차: `물때 보는 법 + 우리 물때 사이트 소개` (기본기 글, 이번 홍보글 활용)
2주차: `대부도 갯벌 체험 물때표`
3주차: `안면도(꽃지·백사장) 낚시 물때`
4주차: `사리·조금 뜻과 이번 달 물때 좋은 날` (월간 물때 캘린더형)

## 6. koreatide.com 쪽에서 같이 해두면 좋은 것 (이미 되어 있는 것 포함)

- ✅ **네이버 사이트 등록**: [네이버 서치어드바이저](https://searchadvisor.naver.com)에
  `koreatide.com` 등록 → 사이트 소유확인 → **사이트맵 제출**(`https://koreatide.com/sitemap.xml`).
  - 소유확인은 **HTML 태그** 방식이 편합니다. 발급된 `content` 값을 Render 환경변수
    **`NAVER_SITE_VERIFICATION`** 에 넣으면 페이지에 자동 삽입됩니다(코드에 이미 연결됨).
- ✅ robots.txt·sitemap.xml 이미 제공 중 (검색봇 허용).
- ⬜ 블로그 글 ↔ 사이트를 서로 오가게: 블로그엔 사이트 링크를, 사이트엔(원하면) 블로그
  링크를 두면 네이버가 "관련성 있는 출처"로 인식하기 쉬움.
