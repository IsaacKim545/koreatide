# 구글 애드센스 처음부터 — koreatide.com 수익화 가이드 (2026 기준)

이 문서 하나로 **계정 개설 → 사이트 등록 → 승인 → 광고 게재 → 수익 지급**까지 순서대로
진행할 수 있습니다. 코드 쪽(로더 스크립트, ads.txt, 개인정보처리방침 광고 고지)은 **이미
사이트에 심어두었습니다.** 여러분은 아래 절차와 환경변수 하나만 챙기면 됩니다.

---

## ⚠️ 먼저: 이 사이트의 승인 가능성 (솔직하게)

2026년 애드센스 심사는 **"콘텐츠(글)"이 있는 사이트**를 강하게 선호합니다. 요건 요약:

- 원본 글 **15~25편 이상**, 편당 본문 **1,000자 이상**, **한 가지 주제 축**이 뚜렷할 것
- 필수 페이지 3종: **소개(About)**, **개인정보처리방침**, **문의(Contact)**
- **HTTPS 필수**(이미 충족), 자체 도메인 권장(koreatide.com 충족)
- 자연 유입 트래픽(공식 최소치는 없지만 일 50~100 방문이 안전선)

koreatide.com 은 지금 **조회 도구(툴) 한 페이지 + 개인정보처리방침** 구조라, 이 상태로
바로 신청하면 **"콘텐츠 부족(low value content)"으로 거절**될 가능성이 높습니다.
그래서 승인 확률을 높이려면 아래 **0단계**를 먼저 하는 것을 강력히 권합니다.

### 0단계 — 승인 전 준비 (권장)
1. **소개(About) 페이지** 추가: 이 서비스가 뭔지, 누가/왜 만들었는지, 데이터 출처(KHOA).
2. **문의(Contact) 페이지** 또는 개인정보처리방침의 문의 이메일을 눈에 띄게(현재
   `itdasoft1@gmail.com` 있음 — 별도 문의 페이지가 있으면 더 안전).
3. **콘텐츠 페이지 쌓기**: 위 네이버 블로그 전략과 같은 글을 **사이트 자체에도** 10편 이상
   올리면 이상적입니다. (예: `/guide/물때-보는-법`, `/guide/대부도-갯벌-물때` …)
   - 최소 대안: "물때 용어 사전", "지역별 물때 안내", "갯벌 안전수칙" 같은 정보 페이지 몇 개.

> 원하시면 소개/문의/가이드 페이지를 코드로 만들어 드릴 수 있습니다(말씀만 주세요).
> 지금 당장 신청해보고 싶다면 그대로 진행해도 되지만, 첫 거절을 감안하세요.

## 1단계 — 애드센스 계정 만들기

1. https://adsense.google.com 접속 → 구글 계정으로 로그인(수익 받을 계정으로).
2. **웹사이트 URL**: `https://koreatide.com` 입력.
3. 국가/지역: **대한민국**, 통화 선택(원). 약관 동의.
4. 계정이 생성되면 **게시자 ID**(`pub-XXXXXXXXXXXXXXXX`)와
   **클라이언트 ID**(`ca-pub-XXXXXXXXXXXXXXXX`)가 발급됩니다. → 이 값을 2단계에 사용.

## 2단계 — 사이트에 애드센스 연결 (코드는 이미 준비됨)

애드센스가 "사이트를 코드로 연결하라"고 하면, 보통 `<head>`에 넣을 스크립트를 줍니다.
**이 사이트는 그 스크립트를 환경변수만 넣으면 자동으로 삽입하도록 이미 구현**돼 있습니다.

- Render 대시보드 → 서비스 → **Environment** → **Add Environment Variable**
  - Key: `ADSENSE_CLIENT`
  - Value: `ca-pub-XXXXXXXXXXXXXXXX` (1단계에서 받은 클라이언트 ID)
- 저장 → 자동 재배포. 그러면 아래가 자동으로 켜집니다:
  - `<head>`에 애드센스 로더 스크립트 + `<meta name="google-adsense-account">` 삽입
  - **`https://koreatide.com/ads.txt`** 가 자동 게시:
    `google.com, pub-XXXXXXXXXXXXXXXX, DIRECT, f08c47fec0942fa0`
- 확인: 배포 후 브라우저에서 `https://koreatide.com/ads.txt` 열어 위 한 줄이 보이면 성공.
  페이지 소스(Ctrl+U)에서 `adsbygoogle.js` 가 보이면 스크립트도 정상.

> 애드센스 화면에서 연결 방식으로 **"AdSense 코드 스니펫"** 또는 **"메타태그"** 를 물으면
> 둘 다 이미 삽입되므로 그대로 **"확인/완료"** 를 누르면 됩니다.

## 3단계 — 사이트 심사 신청

1. 애드센스 → **사이트** → `koreatide.com` 의 상태가 **"검토 중(Getting ready)"** 이 됩니다.
2. Google이 도메인 소유·정책 준수를 검토합니다. **보통 며칠, 길면 2~4주.**
3. 심사 동안 사이트는 정상 운영하고, **콘텐츠를 계속 추가**하세요(도움이 됩니다).

## 4단계 — 승인 후: 광고 켜기 (자동광고 권장)

승인되면(사이트 상태 **"준비됨/Ready"**):
1. 애드센스 → **광고** → **사이트별** → `koreatide.com` → **자동 광고 켜기**.
2. 자동광고는 Google이 페이지를 분석해 알아서 적절한 위치에 광고를 넣습니다
   (별도 광고 단위 코드 없이 동작 — 이미 넣은 로더 스크립트만으로 게재됩니다).
3. 광고 밀도/유형(앵커, 자동 삽입 등)은 자동광고 설정에서 조절하세요. 너무 많으면
   사용자 경험이 나빠지니 **중간 강도**를 권장.

### (선택) 수동 광고 단위를 특정 위치에 넣고 싶다면
자동광고 대신 원하는 자리에 광고를 고정하려면, 애드센스에서 **디스플레이 광고 단위**를
만들어 `data-ad-slot` 값을 받은 뒤 `index.html`의 원하는 위치에 아래를 넣으면 됩니다
(원하시면 제가 그래프 아래/결과 목록 사이에 넣어 드리겠습니다):

```html
<ins class="adsbygoogle" style="display:block"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
     data-ad-slot="0000000000"
     data-ad-format="auto" data-full-width-responsive="true"></ins>
<script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
```

## 5단계 — 수익 지급(PIN·계좌) 설정

1. 수익이 쌓이면 애드센스가 **주소 확인용 PIN**을 우편으로 보냅니다(누적 $10 도달 시).
   받은 PIN을 애드센스에 입력해 주소 인증.
2. **결제 수단**: 애드센스 → **지급** → 은행 계좌 등록(예금주=본인, SWIFT 등).
3. **세금 정보**: 미국 세금 정보(W-8BEN) 입력 요청 시 작성.
4. 잔액이 **$100(지급 기준액)** 을 넘으면 매월 지정일에 자동 지급됩니다.

## 자주 막히는 부분 (체크리스트)

- [ ] 사이트가 **HTTPS**인가? (koreatide.com 은 Render가 자동 발급 — OK)
- [ ] **개인정보처리방침**에 쿠키·제3자 광고(Google) 고지가 있는가? → **이미 반영됨**
      (`/privacy` 5항 "광고 및 광고 쿠키")
- [ ] **소개/문의** 페이지가 있는가? → 소개 페이지 **추가 권장**(현재 없음)
- [ ] **콘텐츠(글)** 가 충분한가? → **가장 큰 관문.** 글/가이드 페이지 확충 필요
- [ ] `ads.txt` 가 정상 게시되는가? → `ADSENSE_CLIENT` 넣으면 자동
- [ ] 금지 콘텐츠·과도한 광고·클릭 유도 문구 없는가?

재신청**하면 됩니다. 재신청 횟수 제한은 없지만, **실질적 개선 없이 반복 신청은 비추천**.
